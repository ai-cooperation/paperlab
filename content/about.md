## 什麼是 論文方法學？

**論文方法學** 是一套 AI 輔助的論文撰寫方法學。

我們不教「研究方法」，我們教的是：**在寫論文的每一個階段，如何用工具與方法來提速與提升品質。**

## 為什麼需要方法學？

整個論文寫作流程的文字都很容易理解，可是很少人有辦法講清楚：**在每一個階段如何處理到位。**

這就是 論文方法學 要解決的問題。

## AI 原生方法 vs 傳統方法

| | 傳統方法 | 論文方法學 |
|---|---------|---------------|
| 文獻搜尋 | 手動搜 Google Scholar | AI 搜尋 + 自動化收集 + 交叉驗證 |
| 文獻整理 | 存在資料夾，再也不打開 | Obsidian 知識圖譜 + 雙向連結 |
| 統計分析 | 背公式、開 SPSS 發呆 | 教你選方法 + 解釋結果 |
| 論文撰寫 | 從空白 Word 開始 | Markdown → 草稿 → 三階段驗證 |
| 圖表製作 | Excel 截圖 | 出版級、高解析向量圖表 |
| 品質審查 | 投出去才知道問題 | 投稿前模擬 reviewer 審查 |
| 期刊格式 | 每換一本期刊重新排版 | 改 YAML 設定，一鍵切換格式 |
| 引用管理 | 手動複製貼上 | Zotero + CrossRef API 驗證 |

## 12 Phase 方法學

從概念探索到審稿回覆，共 12 個階段。每個階段都有：

- **目標**：這一步要完成什麼
- **工具**：用什麼工具
- **操作**：具體怎麼做
- **檢查**：怎麼確認做到位了
