必學工具

AI 時代寫論文的三大基礎設施

Markdown

寫作的通用語言

寫作門檻低，加 YAML 檔頭直接變期刊格式。AI 的輸出就是 Markdown，無縫銜接。

Obsidian

你的第二大腦

知識圖譜讓你看見文獻之間的隱藏關聯。孤立節點 = 可能遺漏的重要文獻。

Zotero

引用的守門員

管理文獻引用 + API 三重驗證，杜絕 AI 給你的假文獻。
