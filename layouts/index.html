必學工具

AI 時代寫論文的三大基礎設施

📝

Markdown

寫作的通用語言

寫作門檻低，加 YAML 檔頭直接變期刊格式。AI 的輸出就是 Markdown，無縫銜接。
🧠

Obsidian

你的第二大腦

知識圖譜讓你看見文獻之間的隱藏關聯。孤立節點 = 可能遺漏的重要文獻。
🛡️

Zotero

引用的守門員

管理文獻引用 + API 三重驗證（CrossRef + Semantic Scholar + OpenAlex），杜絕假文獻。

AI 原生方法 vs 傳統方法

我們不是在傳統方法上「加 AI」，而是從第一步就用 AI 思維設計

環節 傳統方法 Paper Lab 方法
文獻搜尋 手動搜 Google Scholar AI 搜尋 + 自動收集 + 交叉驗證
文獻整理 存在資料夾，再也不打開 Obsidian 知識圖譜 + 雙向連結
統計分析 背公式、開 SPSS 發呆 AI 幫你選方法 + 解釋結果
論文撰寫 從空白 Word 開始 Markdown → AI 草稿 → 多模型校正
品質審查 投出去才知道問題 投之前用 AI 模擬 reviewer 審查
引用管理 手動複製貼上 Zotero + API 三重驗證
